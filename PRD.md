# MdConvert — 产品需求文档 (PRD)

## 1. 产品概述

**产品名称：** MdConvert
**定位：** Markdown 文件查看与格式转换工具
**目标用户：** 无法直接打开 .md 文件的用户（手机端、非技术用户、收到 .md 文件需要查看/分享的人）
**核心价值：** 导入 .md → 渲染为美观的 HTML 页面 → 导出为 PDF/Word

### 1.1 产品愿景

很多平台（GitHub、语雀、Notion）导出的文档是 .md 格式，但普通用户（尤其是手机端）没有打开它的手段。MdConvert 让任何人只需打开网页、导入文件，就能看到排版精美的文档，并一键转为 PDF 或 Word 分享。

### 1.2 核心用户场景

| 优先级 | 场景 | 描述 |
|--------|------|------|
| P0 | 导入查看 | 用户收到 .md 文件，打开网页导入，立即看到渲染后的文档 |
| P0 | 导出 PDF | 用户将渲染后的文档导出为 PDF，用于打印或分享 |
| P0 | 导出 Word | 用户将文档导出为 Word，便于二次编辑或提交 |
| P1 | 在线编辑 | 用户直接在网页上编写 Markdown，实时预览 |
| P1 | 主题切换 | 用户选择不同的渲染主题，适配阅读偏好 |
| P2 | TOC 导航 | 长文档自动生成目录，方便跳转浏览 |

| 层面 | 技术选型 | 说明 |

## 3. 功能需求

### 3.1 主功能：文件导入与渲染

#### F1: Markdown 文件导入

**入口方式（按优先级）：**

1. **文件选择**：点击按钮打开文件选择器，选择 .md / .markdown 文件
2. **拖拽导入**：将 .md 文件拖拽到页面指定区域
3. **粘贴导入**：直接粘贴 Markdown 文本内容

**交互细节：**

- 导入后立即渲染为 HTML，全屏展示渲染结果
- 支持导入本地 .md 文件（FileReader API 读取）
- 文件大小限制：≤ 5MB
- 编码支持：UTF-8（自动检测 BOM）
- 导入失败时显示友好错误提示

#### F2: HTML 渲染展示

- 将 Markdown 完整渲染为语义化 HTML
- 支持 GFM（GitHub Flavored Markdown）：
  - 标题（H1-H6）
  - 加粗、斜体、删除线
  - 有序/无序列表、任务列表
  - 表格
  - 代码块（围栏式和缩进式）
  - 引用块
  - 水平分割线
  - 自动链接
  - 图片
- 代码块语法高亮（highlight.js，190+ 语言）
- 渲染区域支持滚动浏览长文档
- 图片支持相对路径和绝对 URL

#### F3: 导出 PDF

- 点击"导出 PDF"按钮触发
- 将当前渲染的 HTML 内容转为 PDF
- 保留排版样式、代码高亮、表格边框
- 自动分页，避免内容截断
- 文件名默认为导入的文件名（如 `document.pdf`）
- 浏览器端生成，无需后端

#### F4: 导出 Word

- 点击"导出 Word"按钮触发
- 将 HTML 内容转为 .docx 格式
- 保留基本排版（标题、列表、表格、加粗/斜体）
- 代码块以等宽字体呈现
- 文件名默认为导入的文件名（如 `document.docx`）

---

### 3.2 副功能：在线编辑

#### F5: Markdown 编辑器

- 基于 CodeMirror 6 的编辑区域
- Markdown 语法高亮
- 行号显示
- 当前行高亮
- 编辑内容实时同步到预览区

#### F6: 快捷工具栏

编辑模式下显示，提供 Markdown 语法快捷插入：

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| B | 加粗 | Ctrl+B |
| I | 斜体 | Ctrl+I |
| H | 标题 | — |
| " | 引用 | — |
| </> | 代码 | — |
| 🔗 | 链接 | Ctrl+K |
| 📷 | 图片 | — |
| 表格 | 插入表格 | — |
| • | 无序列表 | — |
| 1. | 有序列表 | — |
| — | 分割线 | — |

---

### 3.3 通用功能

#### F7: 主题系统

3 套渲染主题，通过 CSS Variables 切换：

| 主题 | 风格 | 适用场景 |
|------|------|---------|
| Light（默认） | 浅色底，舒适阅读 | 日常查看 |
| Dark | 深色底，低对比度 | 夜间阅读 |
| Print | 高对比度，黑白为主 | 打印预览 |

#### F8: TOC 目录导航

- 从文档自动提取 H1-H3 标题生成目录
- 侧边栏展示，支持层级缩进
- 点击跳转到对应章节
- 长文档时自动出现，短文档时隐藏

#### F9: 状态栏

- 当前主题名称
- 文档字数/行数/字符数统计
